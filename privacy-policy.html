Copy Link First

Privacy Policy

Last updated: 2026-06-02  ·  Version 1.0.0

Overview

This Privacy Policy explains how Copy Link First ("the Extension") handles user data. The Extension is a lightweight Chrome extension that restores the order of the X (Twitter) share menu, placing "Copy link" back above "Send via Chat" so it is no longer tapped by mistake.

The Extension operates entirely within your browser. It makes no network requests and never collects, stores, or transmits any user data to any external server, analytics service, or third party.

1. Data We Process

The Extension does not collect, record, or transmit any personal data. To perform its single function, it only reads — locally, in the page's memory:

  • Share-menu labels — the visible text of the items in X's share dropdown (for example, "Copy link" and "Send via Chat"), read from the page in order to identify which two items to reorder.

This information is used solely to reposition those two menu items. It is never saved, logged, or sent anywhere.

2. Comprehensive Disclosure

2-a. How We Collect, Use, and Store

  • Collect: The Extension collects no data. It only inspects the share menu's existing page elements in your browser to locate the "Copy link" and "Send via Chat" items.
  • Use: The only action performed is visually reordering those two existing menu items so that "Copy link" appears on top. The original click behavior of each item is preserved.
  • Store: The Extension stores nothing — no settings, accounts, identifiers, or browsing history. It uses no cookies and no local storage.
  • Share: Nothing is shared. The Extension makes no external network requests whatsoever.

2-b. Parties With Whom Data Is Shared

We do not share any user data with third parties. The Extension contains no tracking analytics, advertising SDKs, remote servers, or cloud-sync functionality.

3. Permissions Used

  • No special Chrome permissions — The Extension declares no permissions (no storage, downloads, tabs, cookies, or similar access).
  • Content-script host scope: https://x.com/*, https://*.x.com/* — Required solely so the Extension can run on X, find the share menu, and reorder its items. It does not read, modify, or access any other website.

Additional Information

  • Security: All processing is strictly confined to your local browser environment; no data ever leaves your device.
  • Transparency: The Extension is a single content script whose only effect is rearranging two menu items. It requests no broad permissions.
  • Changes: If future feature updates require changes to this policy, the updated date will be reflected at the top of this page.
